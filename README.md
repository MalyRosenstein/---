🚀 הוראות הפעלה לפרויקט
אפליקציית ספר מתכונים קולי (Voice Recipe Book) שנבנתה באמצעות HTML, CSS ו-JavaScript.

🛠 הוראות הרצה
הורידו את קבצי הפרויקט למחשב שלכם.

אתרו את הקובץ בשם index.html בתיקייה הראשית.

לחצו קליק ימני על הקובץ ובחרו "פתח באמצעות" (Open with) -> בחרו בדפדפן מודרני (Chrome, Edge או Firefox).

האפליקציה תיטען ישירות בדפדפן ללא צורך בהתקנת שרת מקומי.

🔐 פרטי התחברות
כדי לגשת לרשימת המתכונים, ניתן להשתמש באחד החשבונות הבאים:

שם משתמש: admin | סיסמה: 123

שם משתמש: user | סיסמה: 456

💡 תכונות מרכזיות
הקראה קולית (TTS): הקראת הוראות המתכון עם אפשרות להגדרת זמן המתנה בין שלב לשלב.

התאמה אישית: תמיכה במצב בהיר (Light Mode) ומצב כהה (Dark Mode) דרך מסך ההגדרות.

ניהול מתכונים: אפשרות להוספת מתכונים חדשים הנשמרים מקומית בדפדפן (LocalStorage).
